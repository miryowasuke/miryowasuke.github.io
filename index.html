English
● 在庫管理 / ストック管理 / 買い物 / 共有

チラストックは、「まだあったっけ？」をなくす、
家族で共有する在庫管理アプリです。

洗剤やトイレットペーパー、食品のストックなど、家の中って意外と「把握しづらい」もの。 チラストックなら、3段階の色分けでサッと確認できて、家族や同居人とも共有できます。 買い忘れや重複購入が減るだけで、買い物がちょっとラクになります。

3段階で直感管理

在庫あり / 残り少 / 在庫なし を色分けでわかりやすく。

共有でズレが減る

「買った？」「まだある？」のやり取りが減ってスッキリ。

補充がスムーズ

在庫切れ時はECサイトへワンタップ（URL登録）で迷わない。

こんな方におすすめ

1
家族で在庫を共有したい
誰かが買った/使い切ったがすぐわかる。
2
買い忘れ・重複購入を減らしたい
買い物前にサッと在庫確認。
3
日用品の補充をラクにしたい
洗剤、紙類、シャンプーなどに最適。

使い方イメージ（3ステップ）

確認 → 共有 → 買い足しをスムーズに。

機能（在庫管理アプリとしてできること）

✓
3段階の在庫管理
在庫状態を直感的に切り替え。家族全員が同じルールで運用できます。
✓
グループ管理
キッチン/洗面所/リビングなど、場所や用途別に在庫を分けて管理。
✓
ECサイト連携
商品にURLを登録しておけば、在庫切れ時に購入ページへ。
✓
商品画像・メモ
写真で一目で判別。補足メモも残せます（サイズ/型番など）。
